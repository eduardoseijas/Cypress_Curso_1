1. npm init #Para iniciar la creacion del proyecto
Con esto crea un package.json

al principio parece como si fuera un error, peros i le das a enter
te da la opcion de colocar descripcion, autor, etc

2. npm install cypress --save -dev #Esta ya es la instalacion de cypress

otra forma de instalar puede ser:
npm cypress install --force #Con esto forzas la instalacion

3. npm install -g npx #otra installacion ES UNA LIBRERIA

para correr cypres y poder iniciar el sistema y abrir la pantalla:
4. node_modules\.bin\cypress open

para correr desde consola o terminal

node_modules\.bin\cypress run (De esta forma corre todos los archivos)

O como instalamos npx, puede abrirse todo con:
 npx cypress open

 para correr desde consola o terminal

  npx cypress run

*Para parar la consola
 Control + C

 Por ejemplo si quiero un comando para correr  desde la consola, un archivo en especifico, debo estar desde la raiz
seria por ejemplo: 
npx cypress run --spec "cypress\e2e\02_Introduccion_JavaScript\01_Variable_Tipo_String.cy.js"
