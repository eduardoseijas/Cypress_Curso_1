luego en mi package.json dentro de scripts debe ir

  "scripts": {
    "cypress:open": "cypress open",
    "cypress:run": "cypress run",
    "test": "echo \"Error: no test specified\" && exit 1",
    "ci": "cypress run --browser chrome --headless",
    "ci:spec": "cypress run --browser chrome --headless --spec"
  }

El comando para lanzalo ya una vez configurado todo es:
  ----------------------------

  cd /d "%WORKSPACE%"
if exist package-lock.json (call npm ci) else (call npm install)
call npx cypress verify
