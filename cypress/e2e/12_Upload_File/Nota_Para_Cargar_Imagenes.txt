Para cargar imagenes hay que cargar un plugin y usar esa libreria

npm install --save-dev cypress-file-upload

y en la carpeta e2e.js coloco la libreria
import 'cypress-file-upload';

y en la carpeta e2e.d.ts coloco la cabecera de la libreria 

/// <reference types="cypress-file-upload" />

####AL MOMENTO DE HACER EL EJERCICIO###
### EJERCICIO 01_Upload_File.cy.js #####
Upload-File
Creo una constante para hacer la ruta

cambiamos el \ por /
const ruta="cypress/e2e/12_Upload_File/Imagen/Imagen_Lobo.jpg"
En este caso es desde la ruta de la carpeta 12

Al usuar una imagen desde la carpeta que yo cree en la doce, uso el .selectFile(ruta, { force: true }) y coloco la ruta completa
